Giới Thiệu về Môn Frontend

Frontend là gì?

Frontend là phần của ứng dụng web mà người dùng tương tác trực tiếp. Nó bao gồm tất cả các yếu tố mà người dùng nhìn thấy và sử dụng, như giao diện người dùng (UI), trải nghiệm người dùng (UX), và các chức năng tương tác.

Các Công Nghệ Frontend Phổ Biến

  • HTML

    Ngôn ngữ đánh dấu siêu văn bản là nền tẳng xây dựng cấu trúc trang web

  • CSS

    CSS là ngôn ngữ dùng để mô tả cách trình bày của tài liệu HTML

  • JavaScript

    JavaScript là ngôn ngữ lập trình giúp tạo ra các trang web động và tương tác

Vai trò của Frontend

Frontend giúp tạo ra một trả nghiệm người dùng mượt mà và dễ dàng tiếp cận. Những nhà phát triển frontend đảm bảo trang web được tương thích trên các thiết bị và máy chủ .
